Image Search Abstraction Layer

User Stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example Input

Enter query to search Google's API below, If you would like to apply an offset then add an optional [,<offset>] to the query.

OR paste the query directly into the browser in the format below:

https://search-images.glitch.me/images/lions?offset=10

Example Output

{extraction: [{url,snippet,thumbnail,context},.....],recentqueries:["lions","bears","lizards",...],offsetQuery: "10"}
